git blame

La fonction "git-blame" montre les dernieres modifications du fichier faites par l'auteur

Syntaxe

git -blame

Affichage

Unknow option: -blame

git blame-L

blame-L sert à annoter des lignes donner, "-L " commence au nulero donner et fini à la fin du fichier et "-L " commence au début du fichier et finit à la ligne donnée.

Syntaxe

git -blame -L 10

Affichage

Unknow option: -blame
